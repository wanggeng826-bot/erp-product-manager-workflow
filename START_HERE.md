# 产品经理工作流 - START HERE

先看 `docs/product-manager-workflow.md`，这是唯一完整入口。

## 你现在怎么开始

1. 打开项目后，先读主工作流文件。
2. 新对话时先告诉我这次是新需求、竞品分析、已有 PRD 生成原型，还是日常讨论。
3. 如果你已经有 PRD，直接上传文件或给我路径。
4. 如果没有 PRD，我先帮你整理需求并产出 PRD。
5. 独立案例只放到 `cases/<case-name>/`。

## 推荐的第一句话

你可以直接复制下面这段发给我：

```text
请按当前项目工作流开始。
如果我已经有 PRD，请先让我上传文件或提供路径，然后读取规范并生成 HTML 可交互原型。
如果我还没有 PRD，请先帮我整理需求并输出 PRD。
```
