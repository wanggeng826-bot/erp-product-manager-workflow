# 产品经理工作流 - START HERE

这个项目已经整理成两个 Skill：`$erp-product-manager` 和 `$ui-optimization-master`。

## 你现在怎么开始

1. 新需求、竞品分析、PRD、原型生成使用 `$erp-product-manager`。
2. UI 评审、原型优化、Ant Design 合规检查使用 `$ui-optimization-master`。
3. 告诉我这次是新需求、竞品分析、已有 PRD 生成原型、UI 优化，还是日常讨论。
4. 如果你已经有 PRD，直接上传文件或给我路径。
5. 如果没有 PRD，我先帮你整理需求并产出 PRD。
6. 独立案例只放到 `cases/<case-name>/`。

## 推荐的第一句话

你可以直接复制下面这段发给我：

```text
请按当前项目工作流开始。
请使用 $erp-product-manager。
如果我已经有 PRD，请先让我上传文件或提供路径，然后读取规范并生成 HTML 可交互原型。
如果我还没有 PRD，请先帮我整理需求并输出 PRD。
```

请使用 $ui-optimization-master 检查这个原型是否符合中文 B 端 ERP 后台和 Ant Design 规范，并给出可执行优化方案。
