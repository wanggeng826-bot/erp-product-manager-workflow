# 新需求方案提示模板

## 1. 用途

本文件用于声明“新需求方案与 PRD”工作流当前生效的主提示词来源。

## 2. 当前生效版本

当前工作流不再优先使用本文件中的默认文案，而是优先读取以下源文件：

- `docs/ui-system/跨境电商IT产品专家（用于新需求方案输出）.md`

该文件现为本项目“新需求方案与 PRD”路径的主提示词。

## 3. 执行要求

进入“新需求方案与 PRD”工作流时，默认按以下顺序执行：

1. 优先读取上述主提示词源文件
2. 再读取：
   - `docs/templates/prd-template.md`
   - `docs/ui-system/ui-interaction-spec.md`
   - `docs/ui-system/erp-reference-patterns.md`
3. 在需求方案和 PRD 输出完成后，继续询问用户是否生成原型

## 4. 兼容说明

如果未来你继续更新“跨境电商IT产品专家”文件，本工作流自动以最新版本为准，本文件无需再重复维护正文。
