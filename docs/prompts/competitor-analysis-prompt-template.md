# 竞品分析提示模板

## 1. 用途

本文件用于声明“竞品分析”工作流当前生效的主提示词来源。

## 2. 当前生效版本

当前工作流不再优先使用本文件中的默认文案，而是优先读取以下源文件：

- `docs/ui-system/竞品分析&产品专家（用于竞品分析及 PRD方案输出）.md`

该文件现为本项目“竞品分析”路径的主提示词。

## 3. 执行要求

进入“竞品分析”工作流时，默认按以下顺序执行：

1. 优先读取上述主提示词源文件
2. 如后续需要继续转产品方案、PRD 或原型，再补充读取：
   - `docs/templates/prd-template.md`
   - `docs/ui-system/ui-interaction-spec.md`
   - `docs/ui-system/erp-reference-patterns.md`

## 4. 框架衔接说明

当前竞品分析源文件已经与本项目规范统一为 `Ant Design`。

因此默认衔接规则如下：

1. 竞品分析阶段，优先遵循源文件本身
2. 如果分析结果继续转产品方案、PRD 或原型，默认继续沿用 `Ant Design`
3. 无需再做额外的 UI 框架切换确认
