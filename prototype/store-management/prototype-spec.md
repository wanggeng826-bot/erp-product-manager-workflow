## 页面清单

| 页面/组件 | 对应文件 | 说明 |
|---|---|---|
| 店铺管理列表页 | index.html | 平台 Tab + 查询区 + 表格 + 操作区 |
| 关联子店铺抽屉 | 抽屉 drawerChildStores | 从主账号店铺的「关联子店铺数」进入，查看该主账号下子店铺列表 |
| 新增店铺抽屉 | 抽屉 modalAddStore | 先录入店铺昵称、邮箱、店铺类型、站点、事业部、负责人、批量权限 → 点击 Shopee 授权 |
| Shopee 授权回调录入弹窗 | 弹窗 modalShopeeAuthResult | 全球账号授权成功后，展示主账号与子店铺资产，要求填写系统别名后再录入 |
| 店铺详情抽屉 | 抽屉 drawerDetail | 基础资料 / 业务信息 / 授权信息 / 操作日志 |
| 编辑基础资料抽屉 | 抽屉 drawerEditBasic | 店铺主体、邮箱、TikTok 广告户 ID / BC ID、基础识别字段 |
| 编辑业务信息抽屉 | 抽屉 drawerEditBiz | 生命周期、多个收款账号、保证金、备注 |
| 重新授权弹窗 | 弹窗 modalReAuth | 确认重新授权 |
| 广告数据更新弹窗 | 弹窗 modalAdSync | TikTok 广告授权成功后设置广告数据更新时段 |

## 交互流程

1. 列表页 → 点击「新增店铺」→ 抽屉
2. 抽屉：先录入店铺基础信息、归属与权限 → 点击「Shopee 授权」
3. 本土店：Shopee 页面选择站点并使用店铺账号登录 → 回调后按已录入信息创建店铺
4. CNSC / 3PF / KRSC：Shopee 页面切换主账号，勾选 Auth Merchant 与站点店铺 → 回调后展示主账号与子店铺资产
5. 用户为所有返回店铺填写系统别名 → 确认录入系统；未填写时不允许确认，用户可取消录入

## 本轮补充范围

- 列表页补齐统一店铺入口字段骨架，并支持平台 Tab 与高级筛选。
- 店铺详情页承载完整资料：基础资料、业务信息、授权信息、操作日志。
- 收款信息从单账号升级为多账号列表，支持新增、删除、保存、下载 CSV。
- 店铺主体变更、收款信息变更在详情页展示字段级变更记录，并通过 Toast 快速告知用户。
- 新增/编辑支持店铺邮箱字段。
- 新增店铺支持批量分配店铺权限。
- TikTok 店铺支持广告户 ID、BC ID。
- TikTok 广告授权成功后弹窗设置广告数据更新时段；已授权广告也可单独设置。
- 收款信息支持列表勾选后批量下载，单店与批量下载均需输入资金信息查看密码。
- 移除页面标题区的「我是管理员 / 我是运营客服」角色切换按钮。
- Shopee 授权不作为独立新增功能，而是新增店铺流程中的授权动作；本土店直接建店，全球账号授权后先进入系统录入确认。
- 全球账号返回子店铺时，必须为每个主账号/子店铺填写系统别名，未填写时阻断确认并提示用户。
- Shopee 主账号店铺在列表展示「关联子店铺数」，点击后以右侧抽屉展示子店铺系统别名、站点、授权状态。
- 主账号授权完成并补齐所有子店铺名称后，系统会同时录入主账号店铺和子店铺，并在主账号行更新子店铺关系。
- 列表或详情页对 Shopee 主账号店铺执行「授权 / 重授权」时，也必须进入主账号授权回调录入弹窗，不能只更新授权有效期。
- 平台专属字段按平台显隐：TikTok 展示广告户 ID / BC ID；Shopee 展示是否主账号授权 / 是否 SIP；其他平台不展示这些字段。
- 授权状态列展示平台授权类型合集：Shopee 展示店铺/广告/联盟，TikTok 与 Amazon 展示店铺/广告，Lazada 与其他平台展示店铺授权。
- 详情页剔除回调 code、Token 状态、权限范围字段。
- 账号类型限定为「自注册 / 购买」，注册类型限定为「企业 / 个人」。
- 浏览器名称使用可维护枚举值选择器，默认枚举「紫鸟 / 战斧」，支持新增与删除枚举值。
- 收款平台使用可维护枚举值选择器，默认枚举「PingPong / WorldFirst / 连连支付」，支持在收款账号行内选择、新增与删除枚举值。
- 列表点击「编辑」时先打开全屏店铺详情，再默认弹出基础资料二级编辑抽屉。
- 抽屉支持点击遮罩空白处关闭；点击抽屉内容区、下拉层、弹窗内容不触发关闭。
