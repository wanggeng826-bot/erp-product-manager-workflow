# 产品经理工作流

## 目录说明

- `AGENTS.md`
  项目级最小路由说明。
- `docs/product-manager-workflow.md`
  主线唯一完整说明。
- `START_HERE.md`
  新用户第一入口。
- `docs/ui-system/`
  UI 规范与提示词源文件。
- `docs/workflows/`
  通用工作流文件。
- `docs/templates/`
  PRD 和原型模板。
- `intake/prd/`
  主线 PRD 入口。
- `cases/<case-name>/`
  独立案例目录。
- `prototype/`
  主线原型输出目录。

## 使用方式

1. 打开 `START_HERE.md`
2. 主线规则以 `docs/product-manager-workflow.md` 为准
3. 独立案例统一放到 `cases/<case-name>/` 内维护
