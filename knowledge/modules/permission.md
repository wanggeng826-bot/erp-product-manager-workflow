## 3. 权限维度

| 维度 | 粒度 | 例 |
|---|---|---|
| 菜单 | 一级 / 二级 | "订单"模块可见 / 不可见 |
| 按钮 | 单个操作 | "批量取消"按钮显隐 |
| 字段 | 字段级 | "联系电话"对财务隐藏 |
| 数据范围 | 行级 | 只能看自己负责的店铺 |

## 4. 业务规则

- 无权限且无需感知 → 隐藏入口
- 有权限但条件不满足 → 显示禁用态 + 解释原因
