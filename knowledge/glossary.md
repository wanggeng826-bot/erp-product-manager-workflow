# Glossary

## 已确认术语

| 术语 | 含义 |
|------|------|
| PRD | 产品需求文档 |
| 原型 | 面向评审和开发理解的页面与交互表达 |
| HTML 可交互原型 | 可在浏览器中打开、点击、查看弹窗/抽屉/状态反馈的原型 |
| 自研 ERP | 公司内部自建的企业资源管理系统 |
| B 端后台 | 面向内部员工或企业用户的业务操作系统 |
| Skill | Codex 可调用的专门能力或工作流 |
| 工作流 | 从用户输入到产出文档、方案、原型或优化建议的一组固定步骤 |
| 知识库 | 跨任务复用的长期背景、偏好、规则和术语 |
| 短期工作记忆 | 当前任务的计划、发现和进度文件 |
| 操作日志 | 记录用户在系统内关键操作行为的日志 |
